# HTML-CSS Proje 
Bu proje HTML, CSS ve jQuery kullanarak geliştirilmiş responsive bir web sitesidir.

## Bu Projede Öğrendiklerim
-Responsive Web Tasarımı 
-Navbar ve Logo Oluşturma 
-Menü Linkleri Oluşturma
-Header Tasarımı
-Görsel ve Overlay Tasarımı
-About Bölümü Oluşturma
-Buton Tasarımı
-Contact Formu Oluşturma
-Footer Tasarımı
-CSS Breakpoint Kullanımı
-Mobil Uyumlu Tasarım
-jQuery ile Menü Butonuna İşlev Ekleme

## Kullanılan Teknolojiler
-HTML5
-CSS3
-jQuery

## Amaç 
Front-End geliştirme sürecinde HTML, CSS ve Responsive Web Tasarımı becerilerimi geliştirmek amacıyla hazırlanmıştır.
